# CS2 Profile Stats
Extension that allows to check cs2 player's statistics right from their steam profile.

## Installation

### Firefox
https://addons.mozilla.org/en-US/firefox/addon/cs2-profile-stats

### Chrome
https://chromewebstore.google.com/detail/cs2-profile-stats/cdjdnohbfomlobiemcceejdnobfgnpje

### Steam

#### Manually
- Open any link in a new tab in your Steam client.
- Enter `https://chromewebstore.google.com/detail/cs2-profile-stats/cdjdnohbfomlobiemcceejdnobfgnpje` and press enter.
- Click `Add to Chrome`, you will be prompted to save the extension file.
- Save the file and the extension is now installed.

#### MIllennium
- Install [Millennium](https://docs.steambrew.app/users/getting-started/installation).
- Install [Extendium](https://steambrew.app/plugin?id=788ed8554492) plugin.
- Open Extendium, click `Install extension`, enter `https://chromewebstore.google.com/detail/cs2-profile-stats/cdjdnohbfomlobiemcceejdnobfgnpje` and click `Install`.

## Self-hosting
- Clone the repository: `https://github.com/CS2-Profile-Stats/cs2-profilestats-extension.git`.
- Chrome: Open `chrome://extensions` from your url bar, click `Load unpacked` and select extension directory.
- Firefox: Uncomment firefox-specific lines in [manifest.json](manifest.json), open `about:debugging` from your url bar, click `Load Temporary Add-on...` and select `manifest.json`.
- Follow instructions in https://github.com/CS2-Profile-Stats/cs2-profilestats-api.

## Features
- View user's Steam data.
- View user's Leetify statistics.
- View user's FaceIt statistics.
- View user's CSStats statistics (soon).
- Compactness.
